# 版本路线图

## 总览

| 版本 | 主题 | 核心目标 |
|---|---|---|
| V0 | 工程基础 | 建立桌面应用骨架、规则系统、安全边界 |
| V1 | 安全清理 MVP | 完成扫描、低风险清理、日志、基础回滚 |
| V2 | 迁移能力 | 完成用户目录迁移、开发工具空间迁移 |
| V3 | 深度分析 | 大文件、重复文件、扫描历史、智能建议 |
| V4 | 产品化发布 | 安装包、便携版、规则更新、稳定性增强 |
| V5 | 高级场景 | Docker、WSL、虚拟机、企业策略，只读分析优先 |

## V0：工程基础

- 初始化 Tauri 2 + React + TypeScript。
- 建立页面框架和基础导航。
- 建立 Rust Native Core 命令结构。
- 建立 `rules/` 规则目录。
- 建立本地日志和配置存储骨架。
- 明确高风险禁区。

## V1：安全清理 MVP

- C 盘空间概览。
- 常见目录扫描。
- 用户临时文件清理。
- Windows 临时文件清理。
- 回收站清理。
- 缩略图和日志类缓存清理。
- 浏览器缓存扫描与清理。
- 操作日志。

## V2：迁移能力

- Windows 默认用户目录迁移。
- Shell Folder 配置读取和修改。
- 开发工具空间管理页面。
- npm、pnpm、bun、cargo、gradle、android 标准迁移方案。
- 环境变量迁移。
- Junction 迁移。
- 迁移回滚。

## V3：深度分析

- 大文件高级筛选。
- 大目录树形分析。
- 重复文件检测。
- 下载目录专项分析。
- 扫描历史对比。
- 智能建议。

## V4：产品化发布

- `.exe` 安装包。
- `.zip` 便携版。
- 可选 `.msi`。
- 崩溃日志。
- 规则版本管理。
- 规则导入导出。
- 应用内更新提示。

## V5：高级场景

- Docker 数据目录只读分析。
- WSL 发行版空间分析。
- 虚拟机镜像分析。
- Android 模拟器镜像分析。
- 游戏库分析。
- 企业策略配置。
