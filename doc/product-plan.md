# 产品方案

## 产品定位

ClearC 是 Windows 系统盘清理与迁移桌面工具，帮助用户治理 C 盘中的系统缓存、临时文件、回收站、用户默认目录、开发工具缓存、AI 工具配置和 SDK 数据。

## 目标用户

- Windows 普通用户。
- 开发者。
- AI 编程工具重度用户。
- Android、前端、Rust、Python、云开发用户。
- C 盘长期空间不足但不希望手动乱删文件的用户。

## 核心能力

| 能力 | 说明 |
|---|---|
| C 盘概览 | 展示总容量、已用空间、可清理空间、可迁移空间 |
| 安全清理 | 清理临时文件、回收站、缩略图、日志、浏览器缓存 |
| 用户目录迁移 | 迁移下载、文档、图片、视频、音乐、桌面 |
| 开发工具空间管理 | 治理 AI 工具、SDK、包管理器、自动化浏览器缓存 |
| 大文件分析 | 展示大文件、大目录和空间来源 |
| 日志与回滚 | 记录操作，支持可回滚操作恢复 |

## 产品能力边界

| 层级 | 能力 | 说明 |
|---|---|---|
| 基础能力 | 扫描、清理、日志 | 最先实现，风险最低 |
| 迁移能力 | 用户目录迁移、开发工具缓存迁移 | 需要预检查和回滚 |
| 高级能力 | Docker、WSL、虚拟机、企业策略 | 先只读分析，后续再评估自动化 |
